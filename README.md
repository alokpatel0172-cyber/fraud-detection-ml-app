# 🔍 Credit Card Fraud Detection - ML Web App

A machine learning web application that detects fraudulent credit card
transactions in real-time using multiple ML algorithms.

## 🌐 Live Demo
[Click here to view the app](https://fraud-detection-ml-app.onrender.com)

## 📌 Problem Statement
Digital payment fraud is a growing threat in India and worldwide.
This project builds an ML model to classify transactions as
**Fraudulent** or **Legitimate** based on transaction features.

## 🧠 ML Algorithms Used
| Algorithm | Type |
|---|---|
| Logistic Regression | Baseline |
| Decision Tree | Tree-based |
| Random Forest ⭐ | Ensemble (Best Model) |
| SVM | Kernel-based |
| KNN | Distance-based |

## 📊 Dataset
- Source: Custom real-world transaction dataset
- Features: Transaction Amount, City, Device OS,
  Payment Gateway, Merchant Category, and more
- Target: `fraud` (0 = Legitimate, 1 = Fraud)

## ⚙️ Tech Stack
- **ML:** Scikit-learn, Pandas, NumPy
- **Web:** Flask, Bootstrap 5
- **Deployment:** Render.com
- **Language:** Python 3
